<p>The original dataset is borrowed from the UCI Human Activity Using Smartphones project (<a href="http://archive.ics.uci.edu/ml/datasets/Human+Activity+Recognition+Using+Smartphones">http://archive.ics.uci.edu/ml/datasets/Human+Activity+Recognition+Using+Smartphones</a>). The dataset is for activity recognition (walking, walking upstairs, walking downstairs, sitting, standing, and laying) from smartphone low level sensor data.</p>

<p>The 'run_analysis.R' script contains five function calls that clean up the original dataset in various ways (See deatils in the Functions section below).</p>

<h2><a id="user-content-file-structure-requirement" class="anchor" href="#file-structure-requirement" aria-hidden="true"><svg aria-hidden="true" class="octicon octicon-link" height="16" version="1.1" viewBox="0 0 16 16" width="16"><path d="M4 9h1v1H4c-1.5 0-3-1.69-3-3.5S2.55 3 4 3h4c1.45 0 3 1.69 3 3.5 0 1.41-.91 2.72-2 3.25V8.59c.58-.45 1-1.27 1-2.09C10 5.22 8.98 4 8 4H4c-.98 0-2 1.22-2 2.5S3 9 4 9zm9-3h-1v1h1c1 0 2 1.22 2 2.5S13.98 12 13 12H9c-.98 0-2-1.22-2-2.5 0-.83.42-1.64 1-2.09V6.25c-1.09.53-2 1.84-2 3.25C6 11.31 7.55 13 9 13h4c1.45 0 3-1.69 3-3.5S14.5 6 13 6z"></path></svg></a>File Structure Requirement</h2>

<p>To call the functions in the run_analysis.R script, the data Human Activity Recognition Using Smartphone Dataset (the extracted 'HCI HAR Dataset' folder) should be located in the same directory as the 'run_analysis.R' file.</p>

<p>e.g., Within the project folder, the following file structure should be kept:</p>

<ul>
<li>run_analysis.R</li>
<li>[HCI Har Dataset]
<dd>-- activity_labels.txt
</dd><dd>-- feactures_info.txt
</dd><dd>-- features.txt
</dd><dd>-- README.txt
</dd><dd>-- [test]<br>
</dd><dd>-- [train]<br></dd></li>
</ul>

<p>The compressed dataset can be downloaded from:
<a href="https://d396qusza40orc.cloudfront.net/getdata%2Fprojectfiles%2FUCI%20HAR%20Dataset.zip">https://d396qusza40orc.cloudfront.net/getdata%2Fprojectfiles%2FUCI%20HAR%20Dataset.zip</a></p>

<h2><a id="user-content-function-call" class="anchor" href="#function-call" aria-hidden="true"><svg aria-hidden="true" class="octicon octicon-link" height="16" version="1.1" viewBox="0 0 16 16" width="16"><path d="M4 9h1v1H4c-1.5 0-3-1.69-3-3.5S2.55 3 4 3h4c1.45 0 3 1.69 3 3.5 0 1.41-.91 2.72-2 3.25V8.59c.58-.45 1-1.27 1-2.09C10 5.22 8.98 4 8 4H4c-.98 0-2 1.22-2 2.5S3 9 4 9zm9-3h-1v1h1c1 0 2 1.22 2 2.5S13.98 12 13 12H9c-.98 0-2-1.22-2-2.5 0-.83.42-1.64 1-2.09V6.25c-1.09.53-2 1.84-2 3.25C6 11.31 7.55 13 9 13h4c1.45 0 3-1.69 3-3.5S14.5 6 13 6z"></path></svg></a>Function Call</h2>

<p>To obtain the submitted tidy data set, users need to follow the following two steps:</p>

<ol>
<li> source the R file: source("run_analysis.R")</li>
<li> call the AvgMerge function: AvgMerge()</li>
</ol>

<p>AvgMerge.txt is the tidy dataset that is submitted for the project.<br>
[Caution: Running the function call can take a couple of minutes.]</p>

<h3><a id="user-content-included-variables" class="anchor" href="#included-variables" aria-hidden="true"><svg aria-hidden="true" class="octicon octicon-link" height="16" version="1.1" viewBox="0 0 16 16" width="16"><path d="M4 9h1v1H4c-1.5 0-3-1.69-3-3.5S2.55 3 4 3h4c1.45 0 3 1.69 3 3.5 0 1.41-.91 2.72-2 3.25V8.59c.58-.45 1-1.27 1-2.09C10 5.22 8.98 4 8 4H4c-.98 0-2 1.22-2 2.5S3 9 4 9zm9-3h-1v1h1c1 0 2 1.22 2 2.5S13.98 12 13 12H9c-.98 0-2-1.22-2-2.5 0-.83.42-1.64 1-2.09V6.25c-1.09.53-2 1.84-2 3.25C6 11.31 7.55 13 9 13h4c1.45 0 3-1.69 3-3.5S14.5 6 13 6z"></path></svg></a>Included Variables</h3>

<p>A total of 68 variables are included in the submitted tidy data set. They are the mean and standard deviation values of the raw accelerometer and gyroscope 3-axial row signals. Only the variables of mean() and std() are included and the weighted average of the frequency components (e.g., meanFreq()) is not included in the final data set. For detailed description regarding variables, see the Variables in the Tidy Data Set section in the codebook.md file.</p>

<h2><a id="user-content-functions" class="anchor" href="#functions" aria-hidden="true"><svg aria-hidden="true" class="octicon octicon-link" height="16" version="1.1" viewBox="0 0 16 16" width="16"><path d="M4 9h1v1H4c-1.5 0-3-1.69-3-3.5S2.55 3 4 3h4c1.45 0 3 1.69 3 3.5 0 1.41-.91 2.72-2 3.25V8.59c.58-.45 1-1.27 1-2.09C10 5.22 8.98 4 8 4H4c-.98 0-2 1.22-2 2.5S3 9 4 9zm9-3h-1v1h1c1 0 2 1.22 2 2.5S13.98 12 13 12H9c-.98 0-2-1.22-2-2.5 0-.83.42-1.64 1-2.09V6.25c-1.09.53-2 1.84-2 3.25C6 11.31 7.55 13 9 13h4c1.45 0 3-1.69 3-3.5S14.5 6 13 6z"></path></svg></a>Functions</h2>
